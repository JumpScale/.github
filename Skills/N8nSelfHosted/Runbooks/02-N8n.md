# 02 -- n8n installeren

## Doel

n8n draaien via Docker Compose met PostgreSQL als database.

---

## Stap 1: Docker Compose bestand aanmaken

```bash
ssh root@<SERVER_IP> "cat > /opt/n8n/docker-compose.yml" << 'EOF'
services:
  n8n:
    image: n8nio/n8n:latest
    container_name: n8n
    restart: unless-stopped
    ports:
      - "127.0.0.1:5678:5678"
    environment:
      - DB_TYPE=postgresdb
      - DB_POSTGRESDB_HOST=n8n-db
      - DB_POSTGRESDB_PORT=5432
      - DB_POSTGRESDB_DATABASE=n8n
      - DB_POSTGRESDB_USER=n8n
      - DB_POSTGRESDB_PASSWORD=${DB_PASSWORD}
      - N8N_HOST=${N8N_DOMAIN}
      - N8N_PORT=5678
      - N8N_PROTOCOL=https
      - WEBHOOK_URL=https://${N8N_DOMAIN}/
      - GENERIC_TIMEZONE=Europe/Amsterdam
      - N8N_ENCRYPTION_KEY=${ENCRYPTION_KEY}
    volumes:
      - n8n_data:/home/node/.n8n
    depends_on:
      n8n-db:
        condition: service_healthy

  n8n-db:
    image: postgres:16-alpine
    container_name: n8n-db
    restart: unless-stopped
    environment:
      - POSTGRES_USER=n8n
      - POSTGRES_PASSWORD=${DB_PASSWORD}
      - POSTGRES_DB=n8n
    volumes:
      - n8n_db:/var/lib/postgresql/data
    healthcheck:
      test: ["CMD-SHELL", "pg_isready -U n8n"]
      interval: 5s
      timeout: 5s
      retries: 5

volumes:
  n8n_data:
  n8n_db:
EOF
```

## Stap 2: Environment bestand aanmaken [Jij moet dit doen]

```bash
ssh root@<SERVER_IP> << 'EOF'
# Secrets genereren
DB_PASSWORD=$(openssl rand -hex 16)
ENCRYPTION_KEY=$(openssl rand -hex 32)

cat > /opt/n8n/.env << ENVEOF
DB_PASSWORD=$DB_PASSWORD
ENCRYPTION_KEY=$ENCRYPTION_KEY
N8N_DOMAIN=<JOUW_DOMEIN>
ENVEOF

chmod 600 /opt/n8n/.env
echo "Environment bestand aangemaakt"
EOF
```

Vervang `<JOUW_DOMEIN>` door je n8n (sub)domein, bijvoorbeeld `n8n.jouwbedrijf.nl`.

## Stap 3: n8n starten

```bash
ssh root@<SERVER_IP> << 'EOF'
cd /opt/n8n
docker compose up -d
docker compose logs -f --tail=30 n8n
EOF
```

Wacht tot je ziet: `n8n ready on 0.0.0.0, port 5678`

## Stap 4: Eerste login [Jij moet dit doen]

n8n maakt een owner-account aan bij de eerste login. Dit gaat via de UI:

```
Ga naar: https://<jouw-domein>/ (na reverse proxy setup)
Of tijdelijk: http://<SERVER_IP>:5678 (als poort open staat)

Maak een account aan met een sterk wachtwoord.
```

## Verificatie

```bash
ssh root@<SERVER_IP> << 'EOF'
echo "=== Containers ==="
docker ps --format 'table {{.Names}}\t{{.Status}}\t{{.Ports}}'
echo "=== Health ==="
curl -s http://localhost:5678/healthz
echo ""
echo "=== n8n versie ==="
docker compose -f /opt/n8n/docker-compose.yml exec -T n8n n8n --version
EOF
```

Verwacht:
- n8n en n8n-db containers draaien
- Health endpoint retourneert `{"status":"ok"}`
